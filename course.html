Course Description

This course is an introduction to web development, covering HTML, CSS, and JavaScript. Students will learn the basics of creating web pages and user interfaces, as well as how to style pages using CSS and add interactivity using JavaScript. By the end of the course, students will have built a small website.

Course Outline

  • Introduction to web development

    • Web development involves creating and maintaining websites. It encompasses various disciplines such as HTML, CSS, and JavaScript. Developers use these technologies to design user interfaces, implement functionality, and ensure compatibility across different devices. Web development plays a crucial role in the online presence of businesses and individuals.
